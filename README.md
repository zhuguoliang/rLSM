### Building on *nix
1. Make sure you have installed the dependencies:

 * rustc 1.31.0 nightly

2. Clone the [source] with `git`:

   ```sh
   $ git clone https://github.com/zhuguoliang/rLSM.git
   $ cd rLSM
   ```
